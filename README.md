#Web-Library backend

---

## Setup
To set up this app you need to:
1. Fill your .env file in format as [template.env](https://github.com/temp-rw/Web-Library/blob/main/WebLibrary/template.env)
2. Run: 
```
   python manage.py makemigrations
   python manage.py migrate
```
